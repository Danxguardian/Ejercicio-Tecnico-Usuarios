## Tecnologias :computer:

Para este proyecto se ocupo el siguiente stack de tecnologias

```
- ReactJS
- Typescript
- React Hooks, React Router Dom, React Custom Hooks
- Redux
- Sagas
- Axios
- Bootstrap
- Styled Components
```
